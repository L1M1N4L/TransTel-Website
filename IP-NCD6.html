Nurse Call Device

IP-NCD6

Transtel IP-NCD6 provides full duplex voice nurse call, intercom and emergency call functions for hospitals or aged care home. IP-NCD6 can be paired with an assortment of NCP to satisfy different requirements. Transtel NCS is design with robustness, easy to use and scalable in mind. The integrated system is centrally powered with easy backup power and can be interfaced with existing hospital system for advanced administrative functions.

Features

  • IP-Based Technology
  • Intuitive User Interface
  • Customizable Alarms and Alerts
  • Two-way Communication
  • Integration Capabilities
  • Wireless Mobility
  • Durability and Hygiene
  • Expandable and Scalable
